## Featured Work

### 1. CURUZA — Fintech Infrastructure for African Retail

Multi-tenant payment and operations platform for hardware retailers. Processes point-of-sale transactions, credit tracking, and business intelligence at scale.

```
Architecture: React 19 → Supabase (PostgreSQL + RLS) → Cloudflare Workers
```

Key capabilities:
- Inventory management with stock movement audit trails
- POS transaction processing with automatic profit reconciliation  
- Debt lifecycle management with payment scheduling
- Role-based access control (Owner/Manager/Employee)
- PDF document generation (invoices, delivery notes)
- Full audit logging for regulatory compliance

### 2. TradeWFriend — Offline-First POS & Payments

Full-screen kiosk POS system for Rwandan SMBs. Designed for tablet-based storefronts with offline resilience and PIN-secured operations.

```
Architecture: Flutter → Riverpod → Supabase → PostgreSQL
```

- SHA-256 secured PIN authentication
- Real-time inventory with low-stock alerts
- Sales processing with profit calculation
- Customer debt tracking with payment plans
- Kiosk lockdown mode for public deployments
- Revenue analytics with 7-day charts
- Light/dark theme with glassmorphism UI

**[View Repository →](https://github.com/hervekubanadev/trade-w-friend)**

- Customer debt lifecycle management
- WhatsApp-integrated payment reminders
